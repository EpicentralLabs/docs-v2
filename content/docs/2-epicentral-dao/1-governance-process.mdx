## Voting Process

The voting process is as follows:

<Steps>
  <Step>

  #### Draft Proposal

  Outline the intent, required resources, and expected outcomes for the idea. This is usually done through a discord forum post in our [Discord](https://discord.gg/5asAuY2sR8) server.

  </Step>
  <Step>

  #### Community Review

  Share the draft for feedback and collect any suggested amendments through the same discord forum post.

  </Step>
  <Step>

  #### Submit On-Chain

  Publish the finalized proposal to the governance program for voting. This is done on-chain through the [Realms](https://realms.today/) user interface. To learn more abou how to create a proposal, please visit their documentation [here](https://docs.realms.today/realms-v2/features/proposals).

  </Step>
  <Step>

  #### Vote & Execute

  Token holders vote, and upon approval the team executes the decision. This is done on-chain through the [Realms](https://realms.today/) user interface. To learn more about how to vote on proposals, please visit their documentation [here](https://docs.realms.today/realms-v2/features/proposals).

  </Step>
</Steps>
